# Snake - 2 Player

A classic Snake game built with Python and Pygame, featuring local 2-player versus mode.

## How to Play

| Player | Controls | Color |
|--------|----------|-------|
| Player 1 | W A S D | Green |
| Player 2 | Arrow Keys | Blue |

- Eat the **red food** to grow and score **10 points**
- Grab the **gold bonus food** when it appears for **50 points** (it disappears after a few seconds)
- Avoid walls, yourself, and the other snake
- The game speeds up as scores increase

Press **Space** to start/restart, **Esc** to quit.

## Requirements

- Python 3
- Pygame

## Setup

```bash
pip install pygame
python snake.py
```
